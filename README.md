# 英語過去問演習

`data/` の問題冊子写真16枚を読み取り、筆記問題をブラウザで演習できるようにした静的アプリです。

公開版：https://shtomi-tech.github.io/english-kakomon/

収録試験：**高校2年生 2026年度 7月8月 理解度確認テスト**

## 収録内容

- 第I問：リスニング8問（MP3音源・自動採点対応）
- 第II問：文法・語法15問、タップ式語句整序5問、同意文完成5問
- 第III問：長文読解7問、記述2問
- 第IV問：長文読解5問、内容説明1問

自動採点47問と、採点基準を使う自己採点1問を収録しています。解答・進捗・最後に開いた大問はブラウザの `localStorage` に保存します。

## 起動

このフォルダで次を実行し、表示されたURLをブラウザで開きます。

```powershell
py -m http.server 8765
```

## 検証

```powershell
node scripts/check-data.mjs
```

## 元資料について

リスニング音源は `data/2026-16-1020.mp3` を使用します。
